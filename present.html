صورت مسئله (نسخه تعاملی)

برنامه‌ای به زبان C بنویسید که مقادیر صورت و مخرج را برای توابع sin و cos و همچنین توان دقت را از کاربر دریافت کرده و سپس نتیجه را با دقت خواسته شده محاسبه و چاپ کند.

شروع کد: فراخوانی کتابخانه‌ها

#include <stdio.h>
#include <math.h>
#include <stdlib.h>

stdio.h: جعبه ابزار استاندارد برای ورودی و خروجی (مثل خواندن از کیبورد و چاپ روی صفحه).

math.h: جعبه ابزار توابع ریاضی پیشرفته مانند `sin`, `cos` و `fabs`.

stdlib.h: شامل توابع عمومی مانند `abs` برای قدر مطلق گرفتن از اعداد صحیح.

تابع اصلی: نقطه شروع برنامه

int main() {
    // ... کدها اینجا قرار میگیرند ...
}

تمام برنامه‌های C از تابعی به نام `main` شروع می‌شوند. این تابع مثل دروازه ورودی شهر است و اجرای برنامه از اولین خط داخل آن آغاز می‌شود.

تعریف متغیرها: ساختن ظرف‌های خالی

double sin_up, sin_down;
double cos_up, cos_down;
int exponent;
int precision;

در این بخش، متغیرهایی (ظرف‌هایی برای نگهداری داده) تعریف می‌کنیم:

double: برای ذخیره اعداد اعشاری با دقت بالا (صورت و مخرج).

int: برای ذخیره اعداد صحیح (توان دقت).

دریافت ورودی سینوس

printf("sin():\n");
printf("up: ");
scanf("%lf", &sin_up);
printf("down: ");
scanf("%lf", &sin_down);

printf: پیامی را روی صفحه چاپ می‌کند تا کاربر بداند چه چیزی را باید وارد کند.

scanf: منتظر می‌ماند تا کاربر عددی را تایپ کرده و Enter را بزند. سپس آن عدد را در متغیر مربوطه ذخیره می‌کند. (`&` به `scanf` آدرس دقیق متغیر را می‌دهد).

دریافت ورودی کسینوس

printf("cos():\n");
printf("up: ");
scanf("%lf", &cos_up);
printf("down: ");
scanf("%lf", &cos_down);

این بخش کاملاً مشابه مرحله قبل است، اما این بار مقادیر صورت و مخرج را برای تابع کسینوس از کاربر دریافت می‌کنیم و در متغیرهای `cos_up` و `cos_down` ذخیره می‌نماییم.

دریافت توان دقت

printf("10 up to (): ");
scanf("%d", &exponent);

در اینجا، از کاربر می‌خواهیم توان دقت مورد نظر را وارد کند (مثلاً 8-).

از فرمت `"%d"` در `scanf` استفاده می‌کنیم، چون `exponent` یک متغیر از نوع عدد صحیح (`int`) است.

پردازش و محاسبه

double sin_angle = sin_up / sin_down;
double cos_angle = cos_up / cos_down;

double sin_val = sin(sin_angle);
double cos_val = cos(cos_angle);

precision = abs(exponent);

۱. ابتدا صورت را بر مخرج تقسیم کرده تا زاویه نهایی را به دست آوریم.

۲. سپس مقادیر سینوس و کسینوس این زوایا را با توابع آماده محاسبه می‌کنیم.

۳. چون دقت چاپ باید یک عدد مثبت باشد، با `abs` قدر مطلق توان ورودی را محاسبه می‌کنیم.

چاپ نتیجه نهایی

printf("\n--- Results ---\n");
printf("sin(%.2f/%.2f) = %.*f\n",
        sin_up, sin_down, precision, sin_val);

printf("cos(%.2f/%.2f) = %.*f\n",
        cos_up, cos_down, precision, cos_val);

در `printf` نهایی از یک ترفند جالب استفاده می‌کنیم:

فرمت `%.*f` به `printf` می‌گوید: "تعداد ارقام اعشار را از یک متغیر جداگانه بخوان". ما متغیر `precision` را به آن می‌دهیم تا نتیجه دقیقاً با دقتی که کاربر خواسته چاپ شود.
